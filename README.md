# Server Templates Catalog

This repository serves as a central hub for various basic server templates. These templates are designed to be ready-to-use so that you don't waste time configuring basic database connections or boilerplate code, allowing you to focus on the actual logic of your project!

## Available Server Templates

The actual templates have been moved to their own dedicated repositories to make cloning and using them easier and more focused. Choose the template that fits your stack:

- **Flask & MongoDB** - A Python base server using Flask and MongoDB. (*[github.com/HarshilD05/Server_Template-Flask-Mongo](https://github.com/HarshilD05/Server_Template-Flask-Mongo)*)
- **Node.js & MongoDB** - A JavaScript/Node.js base server using Express and MongoDB. (*[github.com/HarshilD05/Server_Template-NodeJS-Mongo](https://github.com/HarshilD05/Server_Template-NodeJS-Mongo)*)
- **Ollama & Flask** - A Python base server integrated with Ollama for AI embeddings using Flask. (*[github.com/HarshilD05/Server_Template-Ollama-Flask](https://github.com/HarshilD05/Server_Template-Ollama-Flask)*)
- **Ollama & Node.js** - A Node.js base server integrated with Ollama for AI embeddings. (*[github.com/HarshilD05/Server_Template-Ollama-NodeJS](https://github.com/HarshilD05/Server_Template-Ollama-NodeJS)*)

### How to use these templates

1. Navigate to the repository of the template you want to use.
2. Click the **"Use this template"** button on GitHub to create a new repository based on it, OR clone it directly:
   ```bash
   git clone <repository-url> your-project-name
   ```
3. Follow the specific setup instructions in the template's `README.md`.
